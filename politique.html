Politique de Confidentialité

Bot Discord : TheStarsV2 ™ / Bot TVS™

1. Introduction

La protection de vos données est une priorité pour nous. Cette politique de confidentialité explique quelles données nous collectons via le bot TheStarsV2 ™ / Bot TVS™, comment elles sont utilisées et vos droits en matière de confidentialité.

2. Données Collectées

Lors de l'utilisation de notre bot, nous pouvons collecter les informations suivantes :

  • Identifiants Discord : ID utilisateur, ID serveur, ID message.
  • Contenu des messages uniquement dans le cadre de l'exécution des commandes (exemple : commandes de modération).
  • Statistiques d'utilisation (nombre d'exécutions de commandes, interactions).

Aucune donnée sensible ou personnelle, autre que celles nécessaires au fonctionnement du bot, n'est collectée.

3. Utilisation des Données

Les données collectées sont utilisées uniquement pour les finalités suivantes :

  • Fournir les fonctionnalités du bot (exemple : exécution des commandes).
  • Assurer le bon fonctionnement et la sécurité des services.
  • Améliorer l'expérience utilisateur via des statistiques anonymisées.

4. Partage des Données

Vos données ne seront jamais vendues, louées ou partagées avec des tiers, sauf dans les cas suivants :

  • Conformité avec une obligation légale ou réglementaire.
  • Protection contre les abus ou comportements nuisibles envers le bot ou ses utilisateurs.

5. Stockage des Données

Les données collectées sont stockées sur des serveurs sécurisés. Nous mettons en place des mesures techniques et organisationnelles pour protéger vos informations contre tout accès non autorisé.

Les données sont conservées aussi longtemps que nécessaire pour les finalités mentionnées dans cette politique, sauf si une période de conservation plus longue est requise ou autorisée par la loi.

6. Vos Droits

En tant qu'utilisateur, vous disposez des droits suivants concernant vos données :

  • Accès : Vous pouvez demander une copie des données vous concernant.
  • Suppression : Vous pouvez demander la suppression de vos données.
  • Rectification : Vous pouvez demander la correction de données incorrectes.

Pour exercer vos droits, veuillez nous contacter à l'adresse suivante : kikiX091@outlook.fr.

7. Modifications de la Politique

Cette politique de confidentialité peut être mise à jour à tout moment pour refléter des changements dans nos pratiques ou en raison d'évolutions légales. Toute modification sera notifiée via :

  • Un message sur les serveurs utilisant le bot.
  • Une mise à jour dans la documentation du bot.

Nous vous encourageons à consulter cette page régulièrement.
